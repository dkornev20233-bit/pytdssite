About the game

pyTDS is a Tower Defense game built entirely in Python using Pygame. Running at 1920×1080 and 60 FPS, it features five game modes, 24+ unique tower types, 40+ enemy types across multiple distinct enemy rosters, a full co-op multiplayer system, persistent save files, and many unlockable achievements.

The game has no external engine — all rendering, physics, AI, and networking is hand-coded in Python.
