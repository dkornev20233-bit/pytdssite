Multiplayer

Co-op Fallen mode for 2 players over LAN or internet via ngrok

Multiplayer rules: Fallen mode only. All enemies have +50% HP. Both players place their own towers and share the same wave progression.

How to host

  1. Launch the game → select Multiplayer → Host
  2. The game displays your local IP address and port 7777
  3. Share these with your friend
  4. Wait for client to connect — screen shows a spinner
  5. Once connected, choose the map and start the game
Same network (LAN): Use the displayed local IP directly. No extra tools needed.

Cross-country / internet play (ngrok)

  1. Download ngrok from ngrok.com and create a free account
  2. Run: ngrok tcp 7777
  3. ngrok gives you a forwarding address like 0.tcp.ngrok.io:XXXXX
  4. Share this address with your friend — they use it as the IP and port in the Join screen
Note: Different countries (e.g. UA ↔ RU) will have NAT issues — ngrok or a VPN is required. Free ngrok accounts work fine.

How to join

  1. Launch the game → Multiplayer → Join
  2. Enter the host's IP address and port (default 7777)
  3. Click Connect — wait for the host to start the game

Status overlay

During a multiplayer game a small status badge appears in the bottom-right corner showing:

  • Connection status (● online / ● offline)
  • Your role (HOST / CLIENT)
  • Total messages received
  • Current enemy count
  • Peer unit count
